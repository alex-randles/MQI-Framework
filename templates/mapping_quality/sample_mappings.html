Personnel Mapping

This mapping uses the Friend of a Friend (FOAF) Ontology to transform the name ('NAME'), gender ('GENDER') and location (foaf:based_near) of people. This mapping contains 3 quality issues:

  • Usage of undefined class. foaf:Persons is undefined in FOAF.
  • Usage of undefined property. foaf:first_name is undefined in FOAF.
  • Usage of incorrect range. foaf:based_near requires a range (rr:termType) of an object (rr:IRI) rather than a literal (rr:Literal).

Metric Mapping

This mapping uses the Intent Based Control Loop Ontology (IBCLO) to transform information related to metric data, such as metric name ('__name__') and associated readings ('RESULT_ID'). This mapping contains 3 quality issues:
